# AI Daily Branding

- `ai-daily-logo.svg`: 横版主 logo，用于文档、介绍页、安装器素材。
- `ai-daily-logo-mono.svg`: 横版单色 logo，适合印刷稿或深浅底切换场景。
- `ai-daily-mark.svg`: 桌面图标主 mark，适合窗口、任务栏与 exe 图标来源。
- `ai-daily-mark-mono.svg`: 单色版 mark，适合浅/深底简化场景。
- `favicon.svg`: Web favicon 源文件。
- `ai-daily-icon.png`: 由脚本生成的 256x256 PNG 图标。
- `ai-daily.ico`: 由脚本生成的 Windows `.ico` 图标。

生成 PNG / ICO：

```powershell
.venv\Scripts\python.exe scripts\generate_brand_assets.py
```
